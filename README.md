# webblog-express-mysql

## Installation
To get the Node server running locally:
```bash
git clone https://github.com/luthfikamaal/webblog-express-mysql.git
cd webblog-express-mysql
npm install
npm start
```
Run ```localhost:8000``` in web browser.
